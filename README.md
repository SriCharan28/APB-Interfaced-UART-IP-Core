This repository contains the implementation of a UART 16550 interface with APB bus integration in Verilog, using Xilinx ISE for simulation, Synopsys VC SpyGlass for linting, Synopsys Design Compiler for synthesis.
The UART folder consists of the following :-
1. RTL Folder - Consists of Design code of First In First Out [FIFO] Memory, Transmitter [Tx] Block, Receiver [Rx] Block, Register Block and Top UART Block integrated using the other components.
2. TB Folder - Consists of TestBench code of First In First Out [FIFO] Memory, Transmitter [Tx] Block, Receiver [Rx] Block, Register Block and Top UART Block.
